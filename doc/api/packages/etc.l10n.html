Comment

This directory contains localization enabling files including xml files for describing address formats, currencies and locales as well as localizable properties files for system wide strings. the SRA_ResourceBundle returned by SRA_Controller::getAppResources will contain both the application as well as the platform resources defined in sierra.properties

DTD Summary

address-formats.dtd
 
XML DTD for SIERRA address formats configuration

This xml structure defines how addresses should be formatted for different 
countries

To support validation of your configuration file, include the following
DOCTYPE element at the beginning (after the "xml" declaration):

<!DOCTYPE address-formats PUBLIC "-//SIERRA//DTD ADDRESS FORMATS//EN" 
  "http://sierra-php.googlecode.com/svn/trunk/etc/l10n/address-formats.dtd">

currencies.dtd
 
XML DTD for SIERRA currencies configuration

This xml structure defines how unique SRA_Locale instances should be supported

<!DOCTYPE currencies PUBLIC "-//SIERRA//DTD CURRENCIES//EN" 
  "http://sierra-php.googlecode.com/svn/trunk/etc/l10n/currencies.dtd">
